# Sgeezy

This folder contains a simple static web app to calculate employee tips and allocate cash bills.

Files added:

- [index.html](index.html)
- [script.js](script.js)
- [styles.css](styles.css)

Run locally (open in browser) or use a simple HTTP server:

```bash
python3 -m http.server 8000
# then open http://localhost:8000/index.html
```
